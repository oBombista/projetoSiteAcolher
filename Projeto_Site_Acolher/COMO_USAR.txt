  Igreja Nova Vida de Maricá

📋 INSTRUÇÕES DE USO:

1️⃣ EXECUÇÃO RÁPIDA (Windows):
   - Clique duas vezes no arquivo "instalar_e_executar.bat"
   - O script fará tudo automaticamente!

2️⃣ EXECUÇÃO MANUAL:
   - Abra o terminal/prompt de comando
   - Navegue até a pasta do projeto
   - Execute: pip install -r requirements.txt
   - Execute: streamlit run app.py
   - Abra o navegador em: http://localhost:8501

3️⃣ DADOS DE EXEMPLO:
   - Se não tiver a planilha original, execute: python exemplo_dados.py
   - Isso criará dados de exemplo para teste

📊 FUNCIONALIDADES DO DASHBOARD:

✅ Métricas principais:
   - Total de visitantes
   - Visitas do mês
   - Cidades alcançadas
   - Pessoas sem igreja

✅ Gráficos interativos:
   - Origem dos visitantes
   - Situação religiosa
   - Faixa etária
   - Necessidades identificadas
   - Distribuição geográfica
   - Evolução temporal

✅ Filtros:
   - Por período de datas
   - Por cidade
   - Visualização de dados detalhados

🎨 DESIGN:
   - Paleta de cores do Ministério Acolher
   - Interface moderna e intuitiva
   - Gráficos interativos com Plotly

📁 ARQUIVOS IMPORTANTES:
   - app.py = Aplicação principal
   - requirements.txt = Dependências
   - Cadastro_Visitantes.xlsx = Dados (ou execute exemplo_dados.py)
   - instalar_e_executar.bat = Instalação automática

🆘 PROBLEMAS COMUNS:
   - Se der erro de Python: Instale Python 3.8+ do site oficial
   - Se der erro de dependências: Execute pip install -r requirements.txt
   - Se não abrir: Verifique se a porta 8501 está livre

📞 SUPORTE:
   Para dúvidas, consulte o README.md ou entre em contato com a equipe.
